# Roadmap

## Current Version

- Dependency-free browser console build.
- Dependency-free userscript build.
- Dry-run mode.
- Batch and max-batch limits.
- Local tests for core DOM behavior.
- GitHub Actions check workflow.

## Before Treating As Final

- Validate selectors against the current Instagram desktop activity page.
- Add fixtures for any new Instagram markup variants.
- Decide whether the userscript path is useful or whether console-only distribution is safer.
- Add a manual release checklist with screenshots of expected UI states.
- Consider an explicit pause/resume controller if large deletion runs are needed.

## Out Of Scope

- Credential handling.
- Instagram private API usage.
- Browser extension packaging.
- Automation against accounts other than the current logged-in user.
