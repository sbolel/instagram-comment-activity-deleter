# Security

This repository is private because the utility performs destructive browser automation against a logged-in Instagram session.

## Supported Use

- Only run the generated script from a browser session you control.
- Only delete your own comment activity.
- Do not paste scripts from unreviewed branches into an authenticated browser session.

## Reporting Issues

Open a private issue in this repository with:

- browser and operating system,
- Instagram page URL path,
- sanitized description of the visible controls,
- exact script version or commit.

Do not include Instagram credentials, cookies, session tokens, private messages, or raw account exports.
